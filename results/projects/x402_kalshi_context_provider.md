## Project Description
This context provider uses the Kalshi prediction markets API to deliver RAG services and give AI agents access to prediction market data. It automatically fetches and indexes market data every 24 hours, provides AI-powered search capabilities, and integrates with Claude Desktop via Model Context Protocol. The service uses x402 payment protocol for USDC transactions on EVM networks, enabling monetized access to real-time prediction market insights.
