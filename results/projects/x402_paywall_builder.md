## Project Description
A visual builder for creating customizable x402 protocol paywall pages. Design beautiful, responsive payment pages for crypto micropayments with USDC - no code required.
It's a wrapper around a package that can be used in your projects, the package is a Customizable paywall HTML generator for the x402 protocol. the behaviour of the wall is controlled with meta-tags and headers (keeping in theme with the same concepts of x402 which is expanding the 402 response code behaviour)
